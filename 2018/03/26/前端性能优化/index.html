前端性能优化(1)

发表于 2018-03-26 | 分类于 前端

网页内容

  • 减少Dom数量
  • 减少Http请求次数
  • 减少DNS查询次数
  • 缓存Ajax
  • 减少或者不使用iframe
  • 提前加载
  • 避免404

CSS优化

  • 将样式表置顶
  • 避免CSS表达式
  • 使用link代替@import
  • 注意CSS书写顺序

JavaScript

  • 脚本置底
  • 减少Dom操作，使用变量保存Dom对象
  • javascript代码注意：谨慎使用with,避免使用eval Function函数,减少作用域链查找
  • 将JavaScript和CSS作为外部文件来引用：在实际应用中使用外部文件可以提高页面速度，因为JavaScript和CSS文件都能在浏览器中产生缓存

    图片优化

  • 优化图片
  • 优化CSS Sprites
  • 不要在HTML中使用缩放图片
  • favicon.ico要小而且可缓存
