D3基础：选择集与数据

发表于 2018-05-05 | 分类于 D3.js

数据绑定（Data-join）是D3的核心知识，是D3之所以被称为Data-Driven的原因。它主要是为了解决两个问题：（1）如何根据数据添加元素，（2）当数据更新时，如何修改元素。

1. 选择元素

  • select： 返回匹配选择器的第一个元素
  • selectAll： 返回匹配选择器的所有元素

参数都是CSS选择器：

注意： 符合一个选择器元素的可能有多个，但select只选择第一个。如果要选择符合选择器的所有元素，使用selectAll。

2. 选择集

d3.select和d3.selectAll返回的对象称为选择集（selection），添加、修改、删除属性都得用选择集。选择集还是数据绑定的受体。

2.1 查看状态

  • selection.empty(): 如果选择集为空，则返回true；否则返回false。
  • selection.node(): 返回第一个非空元素，如果选择集为空，返回null。
  • selection.size(): 返回选择集中的元素个数。

2.2 设定和获取属性

  • seletion.attr(name[,value])

    设置或获取选择集的属性。

  • selection.classed(name[,value])

    设定或获取选择集的CSS类，name是类名，value是一个布尔值。

  • selection.style(name[,value[,priority]])

    设定或获取选择集的样式

  • selection.property(name[,value])

    设定或获取选择集的属性，name是属性名，value是属性值。
    有部分属性不能用attr()设定和获取，最典型的就是文本输入框的value属性。

  • selection.text([value])

    设定或获取选择集的文本内容。

  • selection.html([value])

    设定或获取选择集的内部HTML内容。

3. 添加、插入和删除

选择集可以添加、插入、删除元素。

  • selection.append(name)

    在选择集的末尾添加一个元素，name为元素名称。

  • selection.insert(name[,before])

    在选择集中的指定元素之前插入一个元素，name是被插入的元素名称，before是CSS选择器的名称。

  • selection.remove()

    删除选择集中的元素。

4. 数据绑定

数据绑定（Data-Join）是D3处理选择集和数据的方法，是D3之所以被称为D3的原因。

d3.select和d3.selectAll返回选择集，选择集上是没有数据的。数据绑定就是使选择集“拥有”数据，有两个方法：

  • selection.datum([value])

    选择集中的每一个元素都绑定相同的数据value。

  • selection.data(values[,key])

    选择集中的每一个元素分别绑定数组values的每一项。key是一个键函数，用于指定绑定数组时的对应规则。

4.1 datum的工作过程

selection.datum()是将一个数据绑定到选择集的所有元素上，每个元素都拥有相同的数据。

text()的参数是一个无名函数function(d,i)，其中的两个参数分别代表（datum）和索引（index）。

4.2 data的工作过程

selection.data()能将数组各项分别绑定到选择集的各个元素上，并且能指定绑定的规则。

data提供了当数组长度与元素数量不一致时的处理方案：

  • 数组长度>元素数量：为多余数据预留元素位置，以便将来插入新元素
  • 数组长度<元素数量：能获取多余元素的位置，以便将来删除

能够处理这种问题的方法被称为：Data-Join。

与Data-Join相关的三个词：

  • Update: 代表已经存在的元素，将修改这些元素的属性。
  • Enter：代表丢失的元素，它们即将加入选择集，你需要加入它们并赋予它们属性。
  • Exit：代表多余的元素，你将会删除它们。

4.5 update、enter、exit

  • update：修改attr属性
  • enter：给选择集append足够的元素
  • exit：删除多余的元素
